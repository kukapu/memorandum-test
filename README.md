## INICIAR EL PROYECTO

Puedes descargarlo o clonarlo. Una vez dentro del directorio usar el comando 

~~~
npm install
~~~

para instalar todas las dependencia. Despues para iniciar el proyecto 

~~~
npm run dev
~~~
