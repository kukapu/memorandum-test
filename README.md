## INICIAR EL PROYECTO

Puedes descargar o clonar el repositorio. Una vez dentro del directorio usar los siguientes comandos en la terminal:

~~~
npm install
~~~

para instalar todas las dependencia. Despues para iniciar el proyecto 

~~~
npm run dev
~~~
